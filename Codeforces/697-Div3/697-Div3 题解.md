## D

贪心 + 双指针 + 前缀

有些物品价值为1 / 2，每个都有重量，问取价值最少的物品且重量不少于m，问价值。

**题解**：分别对价值为1，2 的进行排序，求前缀和，从0枚举取前几个价值为1的，则可以确定最少取几个价值为2 的，如果满足就取最小值即可。

## E

这个思路很好想

学会了如何求组合数：（用杨辉三角）

```c++
int C[1005][1005];
void init(){
    C[0][0] = 1;
    for(int i = 1;i <= 1000;i++){
        C[i][0] = 1;
        for(int j = 1;j <= i;j++){
            C[i][j] = (C[i - 1][j] + C[i - 1][j - 1]) % MOD;
        }
    }
}
```

这样求C(5,2) 直接用C【5】【2】🆗。
